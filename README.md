## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/scrwww/prj_eamt_btc.git
```

Entre no diretório do projeto

```bash
  cd prj_eamt_btc
```

Instale as dependências

```bash
  npm i
```

Inicie o servidor

```bash
  ionic serve
```
